URL Shortener

A URL shortener written using React, Babel a very basic Node API.

As opposed to storing the generated short url and using that later to query back the full one, the system simply just stores the full url and returns an encoded string corresponding to the id of the newly inserted item as the short url.
This way when querying for the full url from the database (which should probably just be a simple key value store) the short url can be decoded and the query can be run against an (indexed) primary key as opposed to an arbitrary sting.
